# Thai Content Style Guide — DigiWin Thailand

## Register Level
**ภาษาทางการกึ่งสุภาพ** (Semi-formal professional Thai)
- Professional but approachable — not stiff corporate language
- Use ครับ/ค่ะ sparingly (in CTAs or conversational contexts only)
- Avoid slang, but also avoid overly academic Thai

## Translation Rules

1. **Numbers stay identical** — "44 years", "50,000+", "300378" → same digits
2. **Company names stay English** — DigiWin, Thai Alpha Polymer, Ginfong
3. **Product abbreviations stay English** — ERP, MES, WMS, AIoT (spell out Thai meaning on first use)
4. **Shenzhen stock code stays** — 300378
5. **Stats format** — number in English digits, label in Thai
6. **HTML markup preserved** — `<span class="hl-blue">`, `<span data-dw-years>` stay as-is
7. **Emotional arc preserved** — maintain the pain-first → solution → proof → CTA flow
8. **Thai text is ~20-40% longer** — account for in layout testing

## Golden Examples (Reference for Peter's Approval)

### Example 1: Hero Title (Track A)
- EN: `Your True Costs Are <span class="hl-blue">Invisible.</span>`
- TH: `ต้นทุนที่แท้จริงของคุณ <span class="hl-blue">มองไม่เห็น</span>`
- Notes: Direct, impactful. "มองไม่เห็น" = invisible/unseen. Keeps the emotional punch.

### Example 2: Hero Subtitle (Track A)
- EN: `"Shadow Excel" files have replaced your standard operating procedures. Ghost inventory means system says 100, shelf says 50.`
- TH: `ไฟล์ Excel ลับๆ เข้ามาแทนที่ขั้นตอนการทำงานมาตรฐานของคุณแล้ว สินค้าผี — ระบบบอก 100 ชั้นจริงมี 50`
- Notes: "ไฟล์ Excel ลับๆ" = secret Excel files (more natural than literal "Shadow Excel"). "สินค้าผี" = ghost inventory (Thai manufacturing slang).

### Example 3: CTA Button
- EN: `Let's Talk`
- TH: `พูดคุยกับเรา`
- Notes: Warm, inviting. Not "ติดต่อฝ่ายขาย" (Contact Sales) — too cold.

### Example 4: Pain Point Card Title
- EN: `"My team has a workaround for everything"`
- TH: `"ทีมของผมมีวิธีแก้ปัญหาชั่วคราวทุกอย่าง"`
- Notes: Uses ผม (I/my, male polite) — appropriate for factory owner persona.

### Example 5: Stats Label
- EN: `Years Manufacturing Focus`
- TH: `ปีเชี่ยวชาญด้านการผลิต`
- Notes: Concise, no articles needed in Thai.
