# PageBrief: Global Footer

## Build Context
- Built in DIVI Theme Builder (not per-page JSON)
- Built FIRST before any page content
- Persists across all 20 pages

## Specification

| Element | Specification |
|---------|---------------|
| Background | #253B50 (dark navy) |
| Column 1 | DigiWin logo + tagline + "Stock Code: 300378" |
| Column 2 | Quick Links: Products, Industries, Partner Program, About Us |
| Column 3 | Contact: Bangkok office address, phone, email |
| Column 4 | Social media icons (if applicable) |
| Bottom Bar | © 2026 DigiWin Thailand. All rights reserved. · Privacy Policy · Terms |

## Design System
- Background: #253B50
- Text: #FFFFFF (white for dark background)
- Link hover: #3798E4
- Text secondary: rgba(255,255,255,0.7) for subtle text

## DIVI Implementation
- Use DIVI Theme Builder → Global Footer
- 4-column row for main content
- Full-width row for bottom bar
- Text modules for each column
- Social media icons module (if available)

## Reference
- Model after SYSPRO.com footer pattern
- Multi-column layout with links + contact info
- Dark background with white text
